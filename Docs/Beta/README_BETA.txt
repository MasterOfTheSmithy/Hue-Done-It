Hue Done It - Friend Beta README

Hue Done It is a 4-10 player social deduction party game about colorful paint blobs repairing, investigating, and surviving while Bleach/Peroxide impostors sabotage the map.

How to host:
1. Launch the build.
2. Select Create Lobby.
3. Confirm the port, then select Host Lobby.
4. In the lobby, choose settings and wait for players to ready.
5. The host starts the match from the lobby start control.

How to join:
1. Launch the build.
2. Select Join Lobby.
3. Enter the host address and port.
4. Select Join as Client.
5. If the join fails, read the visible status message and include it in a bug report.

Basic objective:
- Color players repair tasks, manage flood pressure, report bodies, vote out Bleach players, and escape the round alive.
- Bleach players try to eliminate Colors, corrupt repairs, manipulate flood pressure, and survive suspicion.
- Paint splats and smears are evidence. Fresh, large, or oddly placed paint can be a clue.

Known beta limitations:
- LAN/direct-IP joining is the expected path for this pass.
- CPU fillers are useful for local testing but are not a substitute for full friend sessions.
- Inventory and secondary killer abilities are still beta systems and may have rough edges.
- Some UI is functional placeholder styling while gameplay stability is prioritized.
- If a match softlocks, use the bug template and attach Player.log when possible.
